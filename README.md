# Projeto 1: Docker CI/CD na AWS (Free Tier)

Este projeto demonstra uma esteira de CI/CD automatizada utilizando GitHub Actions, Docker e AWS.

## 🏗️ Arquitetura do Projeto
![Arquitetura do Projeto](img/arquitetura-profissional.png)

## 🚀 Tecnologias Utilizadas
* **Docker:** Conteinerização da aplicação Node.js.
* **GitHub Actions:** Automação do pipeline (Build e Push).
* **Amazon ECR:** Repositório privado de imagens Docker.
* **Amazon EC2:** Servidor onde a aplicação está hospedada.
